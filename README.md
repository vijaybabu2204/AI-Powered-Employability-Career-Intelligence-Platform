# 🚀 AI-Powered Employability & Career Intelligence Platform

An AI-powered web application that helps students improve their employability by analyzing resumes, predicting placement opportunities, generating personalized learning roadmaps, and providing career insights.

## 📖 Overview

The platform combines Artificial Intelligence, Machine Learning, and Web Technologies to provide students with career guidance and placement assistance.

Users can:

- Upload resumes for AI analysis
- Extract resume information automatically
- View identified technical and soft skills
- Predict placement chances using a Machine Learning model
- Get AI-generated career roadmaps
- View placement opportunities
- Access career intelligence dashboards

## ✨ Features

### 📄 Resume Analyzer
- Upload PDF resumes
- Automatic information extraction
- Skills detection
- Education & experience extraction

### 🤖 AI Career Intelligence
- Personalized career recommendations
- Skill gap identification
- Career roadmap generation

### 📊 Placement Prediction
- Machine Learning based prediction
- Placement probability estimation
- Career readiness evaluation

### 🎯 Career Dashboard
- Student information
- Resume insights
- Placement statistics
- Career recommendations
